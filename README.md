## Usage

This is a project template for [@soche-f2e/vue-cli](http://git.souche.com/souche-f2e/vue-cli).

``` bash
$ npm install -g @souche-f2e/vue-cli --registry=http://registry.npm.souche-inc.com

$ sue init
```
